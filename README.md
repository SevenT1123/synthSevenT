# Simple VST
- One oscillator (sine, saw, square)
- ADSR
- Frequency Modulation
- Filter (Low Pass, By Pass, High Pass)
- Only has one voice

- Created using JUCE Framework
- Require JUCE Library Code and JUCE Modules to run the code
- Visit JUCE: https://juce.com/
